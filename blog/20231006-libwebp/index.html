始终弄不明白这个构建到底怎么用。

官方文档给的方案

网上找到的攻略

这下似乎对了，生成了dll动态链接库。但是架构似乎对不上，而且Command Prompt有5个，我也不确定该用哪个，挨个试一试。

逐个测试

64bit only

x86 Native Tools Command Prompt, ARCH=x86

x86 Native Tools Command Prompt, ARCH=x64
32bit only
